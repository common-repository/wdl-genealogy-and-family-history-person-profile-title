== Description ==


Use the power of WordPress and its database to display the Profile Title

With this plugin you can:

*   Display a profile Image
*   Display First, Middle and Family Names
*   Display Birth Date
*   Display Death Date
*   Edit all Information
*   Change and Customise the Colors to match your site
*   Change Font and Font Size
*   Change the Text Positioning
*   Change the Image Position
*   View a List of profiles including shortcode info


== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.


== Frequently Asked Questions ==

<strong>How Do I Install the Plugin</strong>

Installation of the Plugin is via the standard Wordpress Plugin Installation Process.

*   Download the Plugin: http://warwicklyons.com.au/?ddownload=18
*   Install the Plugin via the WordPress Admin Dashboard Plugins/Add New/Upload
*   Activate the Plugin
*   Once Activated the Admin Menu WDL Genealogy Tools Menu will appear on the Dashboard Menu


<strong>My profile picture does not appear.</strong>

This plugin searches for the profile images in the uploads directory.

You need to to uncheck the Organize my uploads into month- and year-based folders located on the Settings/Media Admin Page
